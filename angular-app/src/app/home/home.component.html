Decentralized sensor data trading network using Hyperledger Composer


This Blockchain application demonstrates sensor data sharing on a network among 'Sellers' on a grid. The sensor data trade is based on exchanging utility tokens for sensor data provided. The network also includes 'Buyers' to trade sensor data with or obtain sensor data from, and 'Exchange' to exchange cash (fiat money) for utility tokens.

Sellers own three assets: raw sensor data, Utility tokens and Cash. They can trade with fellow sellers, exchanges, and buyer(s).

Exchanges on the network own two assets: Cash and Utility tokens. They can trade with sellers, who can buy or sell tokens for cash.
